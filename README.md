## Run Locally

Clone the project

```bash
  https://github.com/jgra1999/balling-store.git
```

Go to the project directory

```bash
  cd balling-store
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```
